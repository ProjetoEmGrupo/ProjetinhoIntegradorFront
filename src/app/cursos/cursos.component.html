Todos os cursos

Cursos Back-end

Java

É uma das linguagens de desenvolvimento mais conhecidas e bem remuneradas do mercado de trabalho. Curso muito didático e que ensinará a dominar programação orientada a objetos e linguagem Java

Phython

Python é uma linguagem de programação de alto nível estruturada e classificadas como orientadas a objetos. Neste curso vou explicar alguns conceitos básicos e mostrar que você também pode aprender a programar!

Kotlin

Nesse curso abordaremos desde os Fundamentos da Linguagem Kotlin, até sua aplicação de forma performática para seu uso no dia a dia. Se você busca um conteúdo que realmente vai trazer uma diferencial na sua carreira.

Cursos Front-end

HTML

é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos. Este é o Curso Completo de HTML5. Primeiro você aprenderá sobre HTML5 desde o básico da estrutura, imagens, semântica e formulário.

CSS

é um código em que você pode fazer alterações rápidas de layout, como definição de cores e fontes, por exemplo. Essa camada proporciona não apenas a facilidade de personalização, como também ajuda a diminuir a repetição de conteúdo na estrutura do código, onde aprenderá tudo isso e muito mais neste curso.

JavaScript

Com este treinamento você será capaz de desenvolver seus códigos javascript para sites e sistemas que rodam no browser. O objetivo é fazer você conseguir desenvolver vários tipos de projetos, validação de formulários e diversos outros exemplos práticos que vão fazer você pensar como um desenvolvedor front-end de verdade.

Cyber Security

Data Secutiry

O Digital Data é o ativo mais importante e precisa ser mantido seguro para que qualquer empresa seja bem sucedida. Aqui, você aprenderá diferentes requisitos e técnicas de segurança de dados, entender quais são os diferentes tipos de dados e a sua importância para diferentes organizações.

Docker

Seja bem-vindo ao curso completo de Docker, ensinaremos todos os recursos da ferramenta, sem precisar de qualquer conhecimento anterior na ferramenta. Neste curso você aprenderá de forma teórica e prática as funcionalidades de Docker, criando projetos e com diversos tipos de ambientes

AWS

É a plataforma de nuvem mais adotada e mais abrangente do mundo. Mesmo que você tenha conhecimento ZERO em AWS, não se preocupe. Esse é o seu curso! Comece a estudar hoje mesmo para a certificação e uma ótima remuneração do Mercado de Tecnologia atualmente.
